# 安装 expo-image-picker

紧急求助功能需要使用 expo-image-picker 来实现图片上传功能。

## 安装步骤

在项目根目录运行以下命令：

```bash
npx expo install expo-image-picker
```

## 配置说明

expo-image-picker 会自动处理以下权限：
- iOS: 相册访问权限、相机权限
- Android: 读取外部存储权限、相机权限

## 功能说明

已实现的功能：
1. ✅ 从相册选择图片
2. ✅ 使用相机拍照
3. ✅ 图片裁剪和压缩（质量80%）
4. ✅ 权限请求和处理
5. ✅ 最多上传3张图片
6. ✅ 图片预览和删除

## 使用方式

用户点击"添加图片"按钮后，会弹出选择框：
- 拍照：直接打开相机拍照
- 从相册选择：打开相册选择已有图片
- 取消：关闭选择框

## 注意事项

1. 在真机上测试时需要授予相应权限
2. Web 平台不支持相机功能，只能选择文件
3. 图片会自动压缩以节省存储和上传流量
