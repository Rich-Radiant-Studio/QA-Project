## 🔄 交互逻辑

### 统计数据点击
```javascript
handleStatPress(stat) {
  switch (stat.label) {
    case '点赞': 显示获赞统计
    case '粉丝': 跳转粉丝列表
    case '关注': 跳转关注列表
    case '朋友': 显示朋友列表
  }
}
```

### 标签页切换
- 点击"我的提问 (56)" → 显示提问列表
- 点击"我的回答 (234)" → 显示回答列表
- 点击"我的收藏 (892)" → 显示收藏内容
  - 默认显示"问题"分类
  - 可切换到"回答"或"评论"分类

### 收藏分类切换
```javascript
setFavoritesTab('questions')  // 显示收藏的问题
setFavoritesTab('answers')    // 显示收藏的回答
setFavoritesTab('comments')   // 显示收藏的评论
```
