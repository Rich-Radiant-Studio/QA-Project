# 回答列表采纳标签更新

## 更新内容

在问题详情页的回答列表中，增加了两种新的采纳标签：

### 1. 作者已采纳标签
- **显示条件**：当回答的 `adopted` 字段为 `true` 时显示
- **样式**：绿色背景，带对勾图标
- **文本**：显示"作者已采纳"
- **位置**：显示在回答卡片的标签区域

### 2. 已采纳数量标签
- **显示条件**：当回答的 `adoptedCount` 字段存在且大于 0 时显示
- **样式**：蓝色背景，带双对勾图标
- **文本**：显示"已采纳 x{数量}"
- **位置**：显示在回答卡片的标签区域

## 数据结构

在回答数据中添加了 `adoptedCount` 字段：

```javascript
{
  id: 2,
  author: '数据分析师小王',
  adopted: false,
  adoptedCount: 3, // 新增：被采纳的次数
  // ... 其他字段
}
```

## 样式说明

### 作者已采纳标签样式
- 背景色：`#f0fdf4`（浅绿色）
- 边框色：`#bbf7d0`（绿色）
- 文字色：`#22c55e`（绿色）
- 图标：`checkmark-circle`

### 已采纳数量标签样式
- 背景色：`#eff6ff`（浅蓝色）
- 边框色：`#bfdbfe`（蓝色）
- 文字色：`#3b82f6`（蓝色）
- 图标：`checkmark-done-circle`

## 显示效果

标签按以下顺序显示：
1. 超级赞徽章（如果有）
2. 作者已采纳标签（如果被采纳）
3. 已采纳数量标签（如果有采纳数）
4. 邀请者标签（如果有）
5. 仲裁相关标签（如果有）
