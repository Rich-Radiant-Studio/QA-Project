# 我的页面浏览历史标签功能说明

## 功能概述

在"我的"页面的内容标签栏中，将"浏览历史"标签放到"我的收藏"旁边，方便用户快速查看浏览记录。

## 功能特点

### 1. 标签位置
- 位置：我的内容区域标签栏，"我的收藏"右侧
- 标签顺序：我的提问 → 我的回答 → 我的收藏 → 浏览历史

### 2. 浏览历史列表

点击"浏览历史"标签后，显示用户最近浏览过的问题列表：

#### 列表项内容
- **问题标题**：浏览过的问题标题
- **作者信息**：问题作者名称
- **浏览时间**：显示"浏览于 X小时前/昨天"等相对时间
- **右箭头**：点击可进入问题详情页

#### 交互功能
- 点击列表项：跳转到对应的问题详情页
- 点击"查看全部"：查看完整的浏览历史

### 3. 数据展示

浏览历史数据包含：

```javascript
{
  id: 1,                                    // 历史记录ID
  title: 'AI大模型会取代程序员吗？',        // 问题标题
  author: 'AI研究员',                       // 问题作者
  time: '1小时前'                           // 浏览时间
}
```

### 4. 测试数据

系统预置了3条浏览历史记录：
1. AI大模型会取代程序员吗？ - AI研究员 - 1小时前
2. 2026年最值得学习的编程语言 - 技术博主 - 3小时前
3. 如何克服社交恐惧症？ - 心理咨询师 - 昨天

## 与原有功能的关系

### 功能菜单中的浏览历史
- 原有的功能菜单中仍保留"浏览历史"入口
- 点击后打开浏览历史弹窗（Modal）
- 弹窗中可以清空历史记录

### 内容标签中的浏览历史
- 新增的标签入口，直接在页面内展示
- 与"我的提问"、"我的回答"、"我的收藏"并列
- 提供快速查看最近浏览记录的方式

## 样式设计

### 标签样式
- 未选中：灰色文字 (#6b7280)
- 选中：红色文字 (#ef4444)，粗体，底部红色指示条

### 列表项样式
- 白色背景
- 底部浅灰色分隔线
- 垂直内边距 14px，水平内边距 16px
- 标题：14px，深灰色 (#1f2937)
- 元信息：12px，浅灰色 (#9ca3af)

## 使用场景

1. **快速回顾**：用户可以快速查看最近浏览过的问题
2. **继续阅读**：方便用户继续阅读之前未读完的内容
3. **查找问题**：帮助用户找回之前看过但忘记收藏的问题
4. **浏览习惯**：了解自己的浏览兴趣和习惯

  '我的收藏 (892)',

### 2. 条件渲染

```javascript
<View style={{ display: activeTab === '浏览历史 (156)' ? 'flex' : 'none' }}>
  {historyList.map(item => (
    <TouchableOpacity onPress={() => handleHistoryPress(item)}>
      {/* 历史记录项 */}
    </TouchableOpacity>
  ))}

```

### 3. 点击处理

```javascript
const handleHistoryPress = (item) => {
  navigation.navigate('QuestionDetail', { id: item.id });
};
```
