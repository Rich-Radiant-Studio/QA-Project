# 收入榜功能说明

## 功能概述

在首页标签栏的"热榜"旁边新增了"收入榜"标签，用户可以查看通过回答问题获得悬赏的金额排行榜。

## 功能特点

### 1. 入口位置
- 位置：首页顶部标签栏，"热榜"和"同城"之间
- 点击"收入榜"标签即可进入收入榜页面

### 2. 排行榜类型
收入榜页面提供三种排行榜：
- **总收入榜**：用户通过回答问题获得的所有悬赏总和
- **月收入榜**：当月获得的悬赏总和
- **周收入榜**：本周获得的悬赏总和

### 3. 排行榜信息展示

每个排行榜项显示以下信息：

#### 基本信息
- **排名**：前3名分别用红色、橙色、黄色徽章标识，其他用灰色
- **用户头像**：48x48 圆形头像
- **用户名**：显示用户昵称
- **认证标识**：蓝色对勾图标（如果已认证）
- **用户简介**：职位和经验年限

#### 统计数据
- **回答数**：用户的总回答数量
- **采纳数**：被采纳的回答数量（绿色图标）
- **悬赏数**：获得悬赏的次数（黄色图标）

#### 收入信息
- **总收入**：大号红色字体显示，超过1万自动转换为"万"单位
- **趋势标识**：
  - 上升趋势：绿色向上箭头 + 百分比
  - 下降趋势：红色向下箭头 + 百分比
  - 持平：灰色横线
- **本月收入**：灰色小字显示当月收入

### 4. 页面功能

#### 顶部导航
- 返回按钮：返回首页
- 标题：显示"收入榜"
- 信息按钮：点击查看收入榜说明

#### 标签切换
- 三个标签可以切换不同时间维度的排行榜
- 当前选中标签显示红色下划线

#### 更新时间
- 显示数据最后更新时间
- 提示"更新于 X分钟前"

#### 列表交互
- 点击任意排行榜项可以跳转到该用户的个人主页
- 滚动查看完整排行榜
- 底部显示"已显示全部排名"

## 数据结构

```javascript
{
  id: 1,                    // 用户ID
  rank: 1,                  // 排名
  user: 'Python老司机',      // 用户名
  avatar: 'url',            // 头像URL
  verified: true,           // 是否认证
  title: '资深Python开发',   // 用户简介
  totalIncome: 125680,      // 总收入（美元）
  monthIncome: 28560,       // 月收入（美元）
  answersCount: 1234,       // 回答数
  adoptedCount: 567,        // 采纳数
  rewardCount: 234,         // 悬赏数
  trend: 'up',              // 趋势：up/down/same
  trendValue: 15            // 趋势百分比
}
```

## 样式特点

- **排名徽章**：前3名使用渐变色（红、橙、黄），其他用灰色
- **收入金额**：红色粗体大字，突出显示
- **趋势标识**：使用颜色和图标直观展示涨跌
- **统计图标**：使用不同颜色的图标区分不同类型的统计数据
- **分隔线**：每个排行榜项之间有浅灰色分隔线

## 使用场景

1. **激励用户**：通过排行榜激励用户积极回答问题
2. **展示收益**：让用户了解平台的收益潜力
