## 📝 注意事项

1. **保持服务器运行**
   - 关闭终端窗口会停止服务器
   - 扫码访问需要服务器持续运行

2. **网络要求**
   - 手机需要联网（不需要同一局域网）
   - 使用 Expo 隧道服务

3. **URL 变化**
   - 每次重启服务器，URL 可能会变化
   - 需要重新生成二维码

4. **性能**
   - 首次加载可能较慢
   - 后续访问会更快

## 🆘 故障排除

### 无法扫码？
1. 确认服务器正在运行
2. 检查二维码是否清晰
3. 确保手机已安装 Expo Go

### 加载失败？
1. 检查手机网络连接
2. 查看服务器终端是否有错误
3. 尝试按 `r` 重新加载

### 需要重新生成二维码？
```bash
node generate-large-qrcode.js
start qrcode-page.html
```
