# 热榜标签功能说明

## ✅ 功能概述

为热榜页面添加了可配置的分类标签功能，支持后台管理维护。

---

## 📱 移动端功能

### 1. 全站热榜标签
**新增标签：**
- 科技数码
- Python编程
- 职场发展
- 健康养生
- 美食烹饪
- 旅游出行

### 2. 所有热榜统一功能
**"全部"标签：**
- 每个热榜类型（全站、国家、行业、个人）都自动包含"全部"标签
- "全部"标签固定在最前面
- 点击"全部"显示该热榜的所有内容
- 点击其他标签显示对应分类的内容

### 3. 标签交互
- 横向滚动查看所有标签
- 点击标签进行筛选
- 选中状态高亮显示（红色背景）
- 未选中状态灰色背景

---

## 🎛️ 后台管理功能

### 访问路径
```
后台管理 → 运营管理 → 热榜标签
URL: /hotlist-tags
```

### 功能特性

#### 1. 热榜类型切换
- 支持切换管理不同热榜类型
- 全站热榜
- 国家热榜
- 行业热榜
- 个人热榜

#### 2. 标签管理
**添加标签：**
- 点击"添加标签"按钮
- 输入标签名称
- 保存后立即生效

**编辑标签：**
- 点击标签卡片的编辑按钮
- 修改标签名称
- 保存更新

**删除标签：**
- 点击标签卡片的删除按钮
- 确认后删除
- 不影响已有数据

#### 3. 标签展示
- 卡片式布局
- 显示标签名称和排序
- 支持编辑和删除操作
- 响应式设计（1/2/3列自适应）

## 📊 数据结构

### 标签配置
```javascript
{
  '全站热榜': ['科技数码', 'Python编程', '职场发展', '健康养生', '美食烹饪', '旅游出行'],
  '国家热榜': ['政策法规', '社会民生', '经济发展', '教育医疗', '环境保护', '基础设施'],
  '行业热榜': ['互联网', '金融', '医疗健康', '教育培训', '房地产', '制造业', '餐饮服务'],
  '个人热榜': ['职业发展', '情感生活', '健康养生', '理财投资', '学习成长', '家庭关系'],
}
```

### 热榜数据
每个标签对应一组热榜数据，包含：
- 排名
- 标题
- 热度值
- 标签（热/新）
- 作者信息
- 回答数
- 趋势（上升/下降）

## 🎨 界面设计

### 移动端
**标签栏：**
- 位置：主标签栏下方
- 样式：横向滚动，圆角按钮
- 颜色：
  - 选中：红色背景 (#ef4444)
  - 未选中：白色背景，灰色边框

**"全部"标签：**
- 固定在最前面
- 默认选中状态
- 与其他标签样式一致

### 后台管理
**页面布局：**
- 顶部：页面标题和说明
- 中部：热榜类型选择器
- 主体：标签卡片网格
- 底部：使用说明

**标签卡片：**
- 白色背景
- 圆角边框
- 悬停阴影效果
- 操作按钮（编辑/删除）

## 📝 使用说明

### 管理员操作流程

1. **访问后台**
   - 登录后台管理系统
   - 点击"运营管理" → "热榜标签"

2. **选择热榜类型**
   - 点击顶部的热榜类型按钮
   - 切换到要管理的热榜

3. **添加标签**
   - 点击"添加标签"按钮
   - 输入标签名称
   - 点击"保存"

4. **编辑标签**
   - 点击标签卡片的编辑图标
   - 修改标签名称
   - 点击"保存"

5. **删除标签**
   - 点击标签卡片的删除图标
   - 确认删除操作

### 用户使用流程

1. **打开热榜页面**
   - 在移动端点击"热榜"

2. **选择热榜类型**
   - 点击顶部标签（全站/国家/行业/个人）

3. **筛选分类**
   - 查看二级标签栏
   - 点击"全部"查看所有内容
   - 点击具体标签查看分类内容
