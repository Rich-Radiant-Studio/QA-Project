# 问题团队列表功能

## ✅ 功能概述

在问题详情页点击团队按钮，先跳转到该问题的团队列表页面，用户可以查看、加入或创建团队，然后再进入具体的团队详情页。

---

## 📱 功能流程

### 用户操作流程
```
问题详情页 → 点击团队按钮 → 问题团队列表页 → 选择团队 → 团队详情页
```

### 详细步骤

1. **在问题详情页**
   - 点击团队图标按钮（people-circle-outline）

2. **进入问题团队列表页**
   - 查看该问题下的所有团队
   - 可以创建新团队
   - 可以通过邀请码加入团队
   - 点击团队卡片进入详情

3. **进入团队详情页**
   - 查看团队信息
   - 参与团队讨论
   - 管理团队成员

---

## 🎨 问题团队列表页面

### 页面结构

#### 1. 头部
- 返回按钮
- 页面标题："问题团队"
- 问题标题（副标题）

#### 2. 操作栏
**创建团队按钮：**
- 红色背景
- 图标：add-circle
- 点击弹出创建团队弹窗

**加入团队按钮：**
- 白色背景，红色边框
- 图标：enter-outline
- 点击弹出输入邀请码弹窗

#### 3. 团队列表
**列表头部：**
- 标题：该问题下的团队 (数量)
- 描述：选择一个团队加入，与大家一起讨论

**团队卡片：**
- 团队头像（56px）
- 团队名称
- 已加入标签（绿色）
- 团队简介
- 成员数量
- 团队长信息
- 活跃度标签（非常活跃/活跃/较活跃）
- 右箭头

#### 4. 空状态
- 图标：people-outline
- 提示文字："该问题下暂无团队"
- 引导文字："点击上方按钮创建第一个团队吧"

---

## 🎯 功能特性

### 1. 团队状态区分
**已加入团队：**
- 显示绿色"已加入"标签
- 点击直接进入团队详情页

**未加入团队：**
- 不显示标签
- 点击弹出确认加入提示
- 确认后加入团队

### 2. 创建团队
**弹窗表单：**
- 团队名称（必填，最多20字）
- 团队简介（选填，最多100字）
- 创建按钮

**创建成功：**
- 显示成功提示
- 关闭弹窗
- 刷新团队列表

### 3. 加入团队
**通过邀请码加入：**
- 输入6位邀请码
- 验证邀请码
- 加入成功提示

**通过列表加入：**
- 点击未加入的团队
- 弹出确认提示
- 确认后加入

### 4. 活跃度标签
**三个等级：**
- 非常活跃（红色）
- 活跃（绿色）
- 较活跃（橙色）

## 💡 使用场景

### 场景一：查看问题相关团队
1. 用户在问题详情页
2. 想了解有哪些团队在讨论这个问题
3. 点击团队按钮
4. 查看团队列表

1. 用户看到感兴趣的团队
2. 点击团队卡片
3. 确认加入
4. 进入团队详情页

2. 点击"创建团队"按钮

1. 用户收到团队邀请码
2. 点击"加入团队"按钮

4. 加入成功
