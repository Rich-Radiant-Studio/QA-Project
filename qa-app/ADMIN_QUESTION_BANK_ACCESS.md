## 📋 操作示例

### 审核用户题库
1. 访问题库管理页面
2. 在状态筛选器选择"待审核"
3. 找到需要审核的题库
4. 点击 ✅ 通过审核 或 ❌ 拒绝
5. 确认操作

### 查看考核统计
1. 访问考核管理页面
2. 查看顶部统计卡片
3. 使用筛选器查看特定数据
4. 点击"查看详情"查看单个考核

### 搜索题库
1. 在搜索框输入关键词
2. 支持搜索：题库名称、作者
3. 实时显示搜索结果

## 🎓 与 Native App 的关联

### Native App 用户操作 → Admin 管理
- 用户上传题库 → Admin 审核题库
- 用户参加考核 → Admin 查看考核记录
- 用户查看历史 → Admin 统计分析

### 数据流向
```
Native App (用户端)
    ↓ 上传题库
Admin (管理端) → 审核
    ↓ 通过
Native App (题库列表) → 用户可见
    ↓ 参加考核
Admin (考核记录) → 数据统计
```

## 📊 数据说明

### 题库状态
- **已通过**：已审核通过，用户可见
- **待审核**：等待管理员审核
- **已拒绝**：审核未通过

### 考核等级
- **优秀**：90-100分（绿色）
- **良好**：80-89分（蓝色）
- **及格**：60-79分（黄色）
- **不及格**：0-59分（红色）
