## 📱 页面功能

### 顶部区域
- **返回按钮**: 返回问题详情页
- **标题**: "相关活动"
- **副标题**: 显示当前问题标题
- **添加按钮**: 发起新活动(右上角)

### 统计栏
显示三个关键数据:
1. **活动总数**: 所有活动数量
2. **报名中**: 正在报名的活动数量
3. **参与人数**: 所有活动的总参与人数

### 活动列表
每个活动卡片包含:
- **活动类型标签**: 线上活动(蓝色) / 线下活动(绿色)
- **状态标签**: 报名中 / 即将开始
- **活动标题**: 活动名称
- **活动描述**: 简短描述(最多2行)
- **时间信息**: 日期和时间
- **地点信息**: 线上会议链接或线下地址
- **发起人信息**: 头像和姓名
- **参与人数**: 当前/最大人数
- **报名按钮**: 立即报名 / 已报名

### 交互功能
1. **点击活动卡片**: 查看活动详情(待实现)
2. **点击报名按钮**: 
   - 未报名 → 已报名(绿色)
   - 已报名 → 取消报名
3. **点击筛选按钮**: 筛选活动(待实现)
4. **点击添加按钮**: 发起新活动(待实现)

## 🔄 使用流程

1. 用户在问题详情页查看问题
2. 点击作者信息区域的活动按钮(日历图标)
3. 跳转到活动列表页面
4. 查看所有相关活动
5. 点击报名按钮参与活动
