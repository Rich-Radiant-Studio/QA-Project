## 🎯 明天可以测试的功能

### 普通成员功能
1. 点击"活动"按钮 → 跳转到活动页面
2. 点击"管理员"按钮 → 弹出申请弹窗
3. 提交申请 → 按钮变为"已申请"
4. 点击"退出"按钮 → 确认退出团队

### 管理员功能
1. 点击"解散"按钮 → 弹出解散确认弹窗
2. 输入团队名称 → 确认解散
3. 查看"审批消息"Tab → 审批加入和管理员申请
4. 点击"同意"或"拒绝" → 处理申请

### 团队列表功能
1. 查看问题团队列表 → 显示用户角色
2. 点击管理员团队 → 进入管理员权限页面
3. 点击成员团队 → 进入成员权限页面
