# 智慧指数系统完整指南

## 🎯 系统概述

## 🔄 完整功能流程

### 流程1: 浏览和管理题库

```
1. 个人中心
   └─> 点击"智慧指数"卡片

2. 智慧指数详情页
   └─> 点击"题库"按钮

3. 题库页面
   ├─> 切换"平台推荐"/"用户上传"标签
   ├─> 浏览题库卡片
   ├─> 点击"开始考核"进入考核
   ├─> 点击"查看详情"查看题库详情
   └─> 点击"上传"按钮

4. 上传题库页面
   ├─> 输入题库名称
   ├─> 添加题目
   │   ├─> 选择题目类型（判断/单选/多选）
   │   ├─> 输入题目内容
   │   ├─> 添加/删除选项
   │   └─> 选择正确答案
   ├─> 点击"提交"按钮
   └─> 返回题库页面
```

### 流程2: 参加智慧考核

```
1. 智慧指数详情页
   └─> 点击"智慧考核"按钮

2. 智慧考核页面（开始前）
   ├─> 阅读考核规则
   └─> 点击"开始考核"按钮

3. 智慧考核页面（答题中）
   ├─> 查看倒计时（10分钟）
   ├─> 查看进度条
   ├─> 选择答案
   ├─> 使用答题卡跳转
   ├─> 点击"上一题"/"下一题"导航
   └─> 点击"提交考核"按钮

4. 提交确认弹窗
   ├─> 查看已答/未答题数
   └─> 点击"确认提交"按钮

5. 考核说明页面
   ├─> 查看考核概况
   ├─> 查看答题统计
   ├─> 查看等级评定
   └─> 点击"查看详情"按钮

6. 考核详情页面
   ├─> 查看概览信息
   ├─> 查看每道题的详细信息
   ├─> 对比用户答案和正确答案
   └─> 点击"返回"按钮
```

### 流程3: 查看考核历史

```
1. 智慧指数详情页
   └─> 点击"考核详情"按钮

2. 考核历史页面
   ├─> 查看统计卡片
   │   ├─> 总考核次数
   │   ├─> 平均分数
   │   └─> 最高分数
   ├─> 浏览考核记录列表
   └─> 点击某条记录

3. 考核详情页面
   ├─> 查看概览信息
   ├─> 查看每道题的详细信息
   └─> 点击"返回"按钮
```

## 📊 6个核心页面详解

### 1. 智慧指数详情页 (WisdomIndexScreen)

**功能**: 展示用户的智慧指数和各项指标

**核心内容**:
- 智慧指数卡片（92.5）
- 4个智慧指标（回答数、采纳数、粉丝数、题目数）
- 平均分数卡片（88.5分）
- 3个操作按钮（智慧考核、考核详情、题库）
- 最近考核记录（最近3条）

**跳转目标**:
- 智慧考核页面
- 考核历史页面
- 题库页面
- 考核详情页面（点击记录）

### 2. 智慧考核页面 (WisdomExamScreen)

**功能**: 进行智慧考核测试

**核心内容**:
- 开始前页面（规则说明）
- 答题页面（10题，10分钟）
- 提交确认弹窗
- 考核说明页面（概况）
- 结果页面（分数、等级）

**特色功能**:
- 倒计时功能
- 答题卡快速跳转
- 自动评分
- 等级评定

### 3. 考核历史页面 (ExamHistoryScreen)

**功能**: 查看所有考核记录

**核心内容**:
- 统计卡片（总次数、平均分、最高分）
- 考核记录列表（题库名称、日期、分数、等级、答对数、用时）

**跳转目标**:
- 考核详情页面（点击记录）

### 4. 考核详情页面 (ExamDetailScreen)

**功能**: 查看某次考核的详细信息

**核心内容**:
- 概览卡片（题库名称、日期、分数、等级、统计、用时）
- 答题详情列表（每道题的完整信息、用户答案、正确答案、对比）

**特色功能**:
- 题目类型标识
- 答案对比说明
- 选项详情展示

### 5. 题库页面 (QuestionBankScreen)

**功能**: 浏览和管理题库

**核心内容**:
- 标签栏（平台推荐、用户上传）
- 题库卡片列表（名称、难度、题数、分类、作者）
- 操作按钮（开始考核、查看详情）
- 上传按钮（右上角）

**特色功能**:
- 类型标识（盾牌/用户图标）
- 难度标签（入门/进阶/高级）
- 分类展示

**跳转目标**:
- 智慧考核页面（点击"开始考核"）
- 上传题库页面（点击"上传"）

### 6. 上传题库页面 (UploadBankScreen)

**功能**: 上传自定义题库

**核心内容**:
- 题库信息（名称、题目数量）
- 题目列表（动态添加/删除）
- 题目卡片（类型、内容、选项、答案）

**特色功能**:
- 三种题型（判断、单选、多选）
- 动态选项管理（2-6个）
- 表单验证
- 自动统计题目数量
