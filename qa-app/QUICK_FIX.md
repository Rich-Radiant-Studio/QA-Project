# ⚡ 快速修复指南

## 🎯 问题
点击补充问题卡片没有跳转到详情页

## ✅ 解决方案 (99%有效)

### 一行命令解决:

```bash
cd qa-app/qa-native-app && expo start -c
```

**就这么简单!** 应用需要完全重启才能加载新的路由配置。

## 📱 测试步骤

1. **重启应用** (使用上面的命令)
2. **打开应用** → 进入问题详情页
3. **点击"补充 (4)"标签**
4. **点击红色测试按钮** (在列表顶部)
5. **应该跳转到补充问题详情页** ✓
