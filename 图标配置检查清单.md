## 图标转换流程

### 后端返回格式

```json
{
  "icon": "fas fa-home"
}
```

### 前端处理流程

```
1. 后端返回: "fas fa-home"
   ↓
2. 保存到 originalIcon: "fas fa-home"
   ↓
3. CategoryIcon 解析:
   - 移除 "fas " → "fa-home"
   - 移除 "fa-" → "home"
   ↓
4. FontAwesome5 显示: <FontAwesome5 name="home" />
   ↓
5. 用户看到: 🏠 图标
```

## 常见问题排查

### Q1: 图标显示为问号或方块

**原因**: 字体文件未正确加载

**解决**:
1. 检查 `build.gradle` 是否有字体配置
2. 清理并重新构建：
   ```bash
   cd android
   .\gradlew clean
   cd ..
   npm run android
   ```

### Q2: 图标全部显示为默认图标

**原因**: 后端返回的图标格式不正确

**解决**:
1. 检查后端返回的数据
2. 查看控制台日志
3. 检查 `originalIcon` 字段

### Q3: 部分图标显示，部分不显示

**原因**: Font Awesome 中不存在该图标名称

**解决**:
1. 检查图标名称是否正确
2. 查看 Font Awesome 5 图标库
3. 使用相似的替代图标
