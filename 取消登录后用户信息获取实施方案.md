# 取消登录后用户信息获取实施方案

# iOS 版本

# 可获取的信息（iOS 5 - iOS 17+）：
✅ IDFV（设备标识符）- 自 iOS 5 起可用，所有版本都支持
    IDFV在恢复出厂设置、卸载所有同一开发者的应用后重装、系统大版本升级时候会变
✅ 设备型号 - 所有版本
✅ 系统版本 - 所有版本
✅ 系统语言 - 所有版本
✅ 时区 - 所有版本
✅ 国家/地区 - 所有版本
✅ 应用版本 - 所有版本
✅ 网络状态（WiFi/蜂窝）- 所有版本

⚠️ 可能受限的信息
    1. IP 地址
        iOS 13-14：✅ 可以获取
        iOS 15+：⚠️ 如果用户开启了 iCloud Private Relay，IP 会被隐藏
        影响：约 5-10% 的用户
        建议：不要依赖 IP 地址
    2. 设备名称
        iOS 13-15：✅ 可以获取用户自定义名称（如"张三的 iPhone"）
        iOS 16+：⚠️ 可能只返回通用名称（如"iPhone"）
        影响：不影响设备识别
        建议：仅用于显示，不用于识别
    3. IDFA（广告标识符）
        iOS 13：✅ 可以获取
        iOS 14+：❌ 需要 ATT 弹窗授权，约 80% 用户拒绝
        建议：不要使用 IDFA


# Android 版本

⚠️可获取的信息（Android 5 - Android 14+）：
✅ Android ID - 所有版本可用（但 Android 8.0+ 是应用级别的，ID会变）
✅ 设备型号 - Pixel 5, Samsung Galaxy S21
✅ 设备品牌 - Google, Samsung, Xiaomi
✅ 制造商 - Google, Samsung
✅ 系统版本 - Android 13
✅ API Level - 33
✅ 屏幕尺寸 - 1080x2340
✅ 屏幕密度 - 2.75
✅ 系统语言 - zh-CN
✅ 时区 - Asia/Shanghai
✅ 国家/地区 - CN
✅ 运营商 - 中国移动
✅ 网络状态 - WiFi/蜂窝
✅ IP 地址 - 192.168.1.100
✅ 应用版本 - 1.0.0
✅ 安装时间 - 时间戳


❌ 已经无法获取的信息
Android 10+ 禁止访问：
❌ IMEI（设备唯一标识）- Android 10+ 完全禁止
❌ 序列号 - Android 8+ 需要特殊权限
❌ MAC 地址 - Android 6+ 已限制，Android 10+ 随机化
❌ 已安装应用列表 - Android 11+ 已限制
